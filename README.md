## 🚀 Installation

This project runs on **OSX, Linux, and Windows**.

1. **Clone the repository:**

   ```sh
   git clone https://github.com/Gugunner/little-lemon-cap-project.git
   cd little-lemon-cap-project
   ```

2. **Install Node packages:**

   ```sh
    npm install
   ```
